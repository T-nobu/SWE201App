# Campus Companion App

## Overview
Campus Companion is a React Native mobile application built using Expo. It helps students quickly access important campus information such as contacts, schedules, and navigation links.

## Features
- Multi-screen navigation (Stack + Bottom Tabs)
- Contacts list with details view
- Weekly schedule display
- Campus map access
- Responsive UI with dynamic styling

## How to Run
1. Install dependencies:
   npm install

2. Start the app:
   npx expo start

3. Run using emulator or mobile device.

## Known Issues
- Uses static sample data
- Map opens in external browser
